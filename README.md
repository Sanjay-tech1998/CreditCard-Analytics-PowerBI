# Credit Card Analytics Dashboard (Power BI)

## Project Overview

This project consists of two interactive Power BI dashboards:\
- Credit Card Customer Report\
- Credit Card Transaction Report

The purpose of this project is to analyze customer behavior, transaction
patterns, and revenue trends using credit card data.

------------------------------------------------------------------------

## Tools & Technologies

-   Power BI Desktop\
-   Power Query (Data Transformation)\
-   DAX (Measures and Calculations)\
-   Data Modeling

------------------------------------------------------------------------

## Dashboard 1: Customer Report

### Objective

To analyze customer demographics and their contribution to revenue.

### Insights

-   Blue card category generates the highest revenue\
-   Customers aged 46--55 contribute the most revenue\
-   High-income group customers generate higher revenue\
-   Male customers contribute slightly more revenue than female
    customers\
-   Swipe transactions are used more frequently than other types

------------------------------------------------------------------------

## Dashboard 2: Transaction Report

### Objective

To analyze transaction trends and spending behavior.

### Insights

-   Highest revenue and transactions occur in Q4\
-   Bills and Entertainment are major expense categories\
-   Online and Swipe transactions dominate usage\
-   Business and white-collar customers generate more revenue
